Privacy Filter Settings

Runs the openai/privacy-filter model entirely in your browser via Transformers.js. No server or API key needed.
First run downloads the model (~30–50 MB, cached afterwards). WebGPU is used when available, with WASM fallback.
Detects: names, emails, phones, addresses, URLs, dates, account numbers, secrets.
